ICU生の時間割・履修計画アプリの決定版

## できること
### 🔎あなたにピッタリのコースを見つける
- 公式授業一覧から取得されたすべてのコースを検索
### 📅完璧な時間割を組む
-  Long 4, 5, 6, 7など、ICUならではのコマ割りも表示
### 🔄マルチデバイス同期
-  ログインしてすべてのデバイスで最新の時間割を共有
